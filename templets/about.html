About This Project

AI House Price Prediction is a Machine Learning project that predicts house prices based on important property features.

How It Works

The system uses a Random Forest Regression model trained using house price data.

Features Used

  • Living Area
  • Number of Bedrooms
  • Number of Bathrooms
  • Year Built

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-learn
  • HTML
  • CSS
